# Milter

This software originated as a clone of ppymilter (http://code.google.com/p/ppymilter/) by Hirohisa Mitsuishi (https://github.com/bongole).

## Installation

Add this line to your application's Gemfile:

    gem 'milter'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install milter
